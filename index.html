Returns diagnosis, confidence, risk level + disclaimer. Risk Level Logic
def get_risk_level(confidence, diagnosis): if diagnosis == "NORMAL": return "LOW" if confidence >= 0.85: return "HIGH" if confidence >= 0.65: return "MEDIUM" return "LOW"
